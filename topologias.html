TOPOLOGÍAS DE RED

Características y aplicaciones


Topología de Bus

- Ventajas: Sencilla y económica; fácil de instalar.

- Desventajas: Problemas en la red afectan a todos los dispositivos; difícil de expandir.

Topología de Estrella

- Ventajas: Facilita la gestión y solución de problemas; cada dispositivo está conectado al hub central.

- Desventajas: Dependencia del hub central; puede ser costosa si se usan muchos cables.
Bus y Estrella


La topología híbrida combina varias topologías, como estrella con bus o anillo, para aprovechar sus ventajas.

- Combinaciones: La topología híbrida combina elementos de varias topologías, como estrella con bus o anillo con estrella.

- Usos: Se utiliza en redes grandes y complejas, donde se necesitan las ventajas de diferentes topologías para manejar múltiples dispositivos y áreas. Ideal para empresas y organizaciones que requieren flexibilidad y escalabilidad.
Híbrida


Topología de Anillo

- Ventajas: Datos circulan en un solo sentido, evitando colisiones; funciona bien para redes con alta carga de datos.

- Desventajas: Si un dispositivo falla, afecta toda la red; difícil de agregar nuevos dispositivos.

Topología de Malla

- Ventajas: Alta redundancia y fiabilidad; cada dispositivo tiene múltiples conexiones.

- Desventajas: Costosa y compleja de instalar; requiere mucho cableado.
